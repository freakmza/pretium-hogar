# PRETIUM HOGAR

Aplicacion para registrar y entender consumos del hogar: compras, servicios, impuestos, escuela, vencimientos y reportes mensuales.

## Estructura

- `web/`: app web principal, funciona abriendo `index.html` en el navegador.
- `android/`: base Android WebView para empaquetar la app como APK.
- `docs/`: idea funcional y alcance de la primera version.

## Primera version

- Guardado local en el navegador.
- Presupuesto opcional, apagado por defecto.
- Compras, servicios, escuela, vencimientos y reportes.
- Exportar e importar datos en JSON.
- Tema claro y oscuro.

## Probar

Abrir:

`web/index.html`

O levantar un servidor local desde `web/`:

`php -S 127.0.0.1:8080`
